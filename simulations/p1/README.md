Bouncing Ball Simulation
======

How to run:
* Open index.html
* Enjoy

Used libraries:
* Three.js (for displaying 3D objects)
* jQuery (controls)
* Bootstrap (style)

Additional features:
* 3D viewport controls (360 degree view, zoom in / out)
* User-defined coefficients (steps per frame, elasticity, friction, etc.)
* Pyramid container
* Multiple balls
* A nice "throw" button
